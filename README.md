# Organizador de Imágenes y Archivos

Aplicación para organizar imágenes y otros archivos en carpetas numeradas, por fecha de creación.

## Estructura del Proyecto

- `core/` - Lógica principal de la aplicación
- `gui/` - Interfaz de usuario
- `assets/` - Recursos estáticos
- `dist/` - Archivos generados

## Uso

Ejecutar `python main.py` para iniciar la aplicación.

### Para convertir a exe usar el siguinte comando:

pyinstaller --onefile --windowed --name "OrganizadorImagenes" --icon="assets/icono.ico" main.py
